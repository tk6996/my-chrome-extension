# Password Generator

Chrome extension สำหรับสร้างรหัสผ่านที่ปลอดภัย กำหนดความยาวและชนิดตัวอักษรได้เอง

## วิธีติดตั้ง (Load Unpacked)

1. เปิด Chrome แล้วไปที่ `chrome://extensions`
2. เปิดสวิตช์ **Developer mode** (มุมขวาบน)
3. คลิก **Load unpacked**
4. เลือกโฟลเดอร์ `password-generator-extension` (โฟลเดอร์นี้)
5. ไอคอนของ Password Generator จะปรากฏในแถบเครื่องมือ (toolbar) — หากไม่เห็น ให้กดไอคอนรูปจิ๊กซอว์ (Extensions) แล้ว pin ไว้

## วิธีใช้งาน

1. คลิกไอคอน extension เพื่อเปิด popup
2. ตั้งค่าความยาวรหัสผ่านและชนิดตัวอักษรที่ต้องการ (ตัวพิมพ์ใหญ่/เล็ก, ตัวเลข, สัญลักษณ์)
3. กดปุ่มสร้างรหัสผ่าน แล้วคัดลอกไปใช้งานได้ทันที (ใช้สิทธิ์ `clipboardWrite`)

## อัปเดตโค้ด

หลังแก้ไขไฟล์ในโฟลเดอร์นี้ ให้กลับไปที่ `chrome://extensions` แล้วกดปุ่ม **รีโหลด (⟳)** ที่การ์ดของ extension เพื่อให้ Chrome โหลดโค้ดใหม่
